# Days of April · HR Consultancy — media host

Public media for the auto-post kit. Instagram fetches post images/reels from here via
jsDelivr (`https://cdn.jsdelivr.net/gh/Hashkodi/days-of-april-media@main/<slug>/…`).

Only content that is being posted publicly anyway lives here — never brand internals or
client data. Files are laid out one folder per post slug by `scripts/build_media.py`
in the (private) kit repo.
